Troubleshooting

Restrictions aren't working

If your restrictions aren't working, please check the following:

1. (iPhone/iPad users) Disable Share Across Devices

  • If Screen Time is shared across devices, Apple has reported that it may cause instability
  • We recommend disabling "Share Across Devices" in iPhone Settings → Screen Time
  • We recommend disabling "Share Across Devices" in Mac System Settings → Screen Time
  • Even if you disable this setting on iPhone, if it remains enabled on Mac, "Share Across Devices" will be re-enabled. You need to disable it on all devices.

2. (iPhone/iPad users) Verify Screen Time permissions

  • Go to iPhone "Settings" → "Screen Time" → "Applimitsts" and verify permissions are enabled
  • If disabled, grant permission again from the Applimits app

3. (iPhone/iPad users) MDM users

  • On devices with MDM control (Google Device Policy, Intune, etc.), the MDM sync timing may cause issues with the app's time limit settings
  • If issues occur, please tap "Re-register All Schedules" in the settings screen

4. Re-register schedules

  • If you changed your device's Screen Time or other settings, the app's settings may be removed
  • Please tap "Re-register All Schedules" in the settings screen to re-apply

5. Check schedule settings

  • Are the target days enabled?
  • Are the time limits set correctly?
  • Is "Disable for today" enabled?

If the problem persists, please contact us through the contact form.

Restrictions stopped working after an OS update

If restrictions stop working after an OS update, try the following:

1. Restart your device

  • Completely restart your device

2. Re-register schedules

    Applimits
  • Open Applimits Settings screen
  • Tap "Re-register All Schedules"
  • All schedules and Downtime settings will be re-applied

3. Re-verify Screen Time permissions

  • Go to iPhone "Settings" → "Screen Time" → "Applimits" and verify permissions
What does "Re-register All Schedules" do?

"Applimitsster All Schedules" re-applies all schedule and Downtime settings.

When to use:

  • When restrictions aren't working properly
    • Applimitster an OS update

How to do it:

  1. Open the Settings screen
  2. Tap "Re-register All Schedules"
  3. Tap "OK" in the confirmation dialog

Note: This operation re-applies all schedule and Downtime settings. Your app settings are preserved.

(iPhone/iPad) Prevent app deletion

To prevent deleting ApLimit itself, use iPhone/iPad Screen Time settings:

  1. iPhone Settings > Screen Time
  2. Content & Privacy Restrictions
  3. iTunes & App Store Purchases
  4. Set "Deleting Apps" to "Don't Allow"

Basics

What is ApLimit?

ApLimit is an iOS app designed to help you manage your smartphone usage and improve focus.

Key Features:

  • Time Limits: Set daily usage limits for specific apps or categories.
  • Downtime: Block apps completely during specific hours (e.g., bedtime).
  • Focus Timer: Use the Pomodoro technique to stay concentrated.
  • Hard Limitation: A powerful mode that prevents removing restrictions until the next day.
Difference between Time Limits and Downtime?

Time Limits:
Limits the total amount of time you can use apps per day (e.g., "1 hour total"). You can use them whenever you want until the time runs out.

Downtime:
Blocks apps during a specific time period (e.g., "10:00 PM to 7:00 AM"). Perfect for ensuring uninterrupted sleep or study time.

What's included in the free version?

The free version includes:

  • Time Limits: Create up to 1 limit
  • App Selection: Select up to 2 apps/categories per limit
  • Downtime: Unlimited settings

Upgrade to Pro for unlimited limits, unlimited app selection, Focus Timer, and Hard Limitation features.

Features

What is Hard Limitation?

Hard Limitation is a feature that, once enabled, cannot be disabled until the next day.

It prevents you from giving in to temptation and removing restrictions "just for a minute."

Note: Please be careful when enabling this. You will not be able to change or delete the settings until tomorrow.

How to use Focus Timer?

The Focus Timer uses the Pomodoro technique (e.g., 25 min focus, 5 min break) to boost productivity.

  1. Go to the "Focus" tab
  2. Select apps to block
  3. Set the duration (15 min+)
  4. Tap "Start"

Selected apps will be blocked while the timer is running. If Hard Limitation is enabled, you cannot stop the timer.

What is "Disable for today"?

This feature allows you to temporarily turn off a specific limit for the rest of the day.

Useful if you need to use an app for an emergency. The limit will automatically resume tomorrow.

Subscription

How to restore purchases?

If you have a Pro plan and changed devices:

  1. Go to Settings
  2. Tap "Upgrade to Pro"
  3. Tap "Restore Purchases"

Ensure you are signed in with the same Apple ID used for purchase.

How to cancel subscription?

Manage subscriptions via iPhone Settings:

  1. Settings App > Tap your name at the top
  2. Subscriptions
  3. Select ApLimit
  4. Tap "Cancel Subscription"
