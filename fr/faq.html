Dépannage

Les restrictions configurées ne fonctionnent pas correctement

⚠ Pour les utilisateurs d'iOS 26

Un bug a été signalé sur iOS 26 où les limites de temps d'utilisation (limites d'app Temps d'écran) ne fonctionnent pas correctement. Pour plus de détails, veuillez consulter le Apple Developer Forum.

Si ce problème se produit fréquemment, veuillez envisager d'utiliser la « Restriction programmée » ou « Restreindre maintenant » en attendant la correction du bug.

Si les restrictions ne fonctionnent pas, veuillez vérifier les points suivants :

1. (Utilisateurs iPhone/iPad) Désactiver le partage entre appareils

  • Apple a signalé que le partage de Temps d'écran entre appareils peut provoquer une instabilité
  • Nous recommandons de désactiver « Partager entre les appareils » dans Réglages iPhone → Temps d'écran
  • Nous recommandons de désactiver « Partager entre les appareils » dans Réglages Mac → Temps d'écran
  • Même si vous désactivez ce réglage sur iPhone, si le réglage reste activé sur Mac, le « Partage entre les appareils » peut se réactiver. Nous recommandons de le désactiver sur tous les appareils

2. (Utilisateurs iPhone/iPad) Vérifier les autorisations Screen Time

  • Allez dans Réglages iPhone → Temps d'écran → Applimits et vérifiez que les autorisations sont activées
  • Si les autorisations sont désactivées, veuillez les accorder à nouveau depuis l'application Applimits

3. (Utilisateurs iPhone/iPad) Utilisateurs MDM

  • Sur les appareils gérés par MDM comme Google Device Policy ou Intune, la synchronisation MDM peut causer des problèmes avec les paramètres de temps d'utilisation de l'application
  • En cas de problème, veuillez exécuter « Réenregistrer les plannings » depuis l'écran des paramètres

4. Réenregistrer les plannings

  • Si vous modifiez les paramètres de Temps d'écran ou d'autres paramètres de l'appareil, les paramètres de l'application peuvent être supprimés.
  • Veuillez appuyer sur « Réenregistrer les plannings » dans l'écran des paramètres pour réappliquer vos paramètres

5. Vérifier les paramètres de planning

  • Les jours de la semaine ciblés sont-ils activés ?
  • La limite de temps est-elle définie sur autre chose que 0 minute ?
  • L'option « Désactiver pour aujourd'hui » est-elle activée ?
Les restrictions ne fonctionnent plus après une mise à jour de l'OS

Si les restrictions ne fonctionnent plus après une mise à jour de l'OS, veuillez essayer les étapes suivantes :

1. Redémarrer l'appareil

  • Redémarrez complètement votre appareil

2. Réenregistrer les plannings

  • Ouvrez l'écran des paramètres d'Applimits
  • Appuyez sur « Réenregistrer les plannings »
  • Tous les plannings et paramètres de temps d'arrêt seront réappliqués

3. Revérifier les autorisations Screen Time

  • Allez dans Réglages iPhone → Temps d'écran → Applimits et vérifiez les autorisations
Qu'est-ce que « Réenregistrer les plannings » ?

« Réenregistrer les plannings » est une fonction qui réapplique tous les plannings et paramètres de temps d'arrêt.

Quand l'utiliser :

  • Lorsque les restrictions ne fonctionnent pas correctement
  • Après une mise à jour de l'OS

Comment l'exécuter :

  1. Ouvrez l'écran des paramètres
  2. Appuyez sur « Réenregistrer les plannings »
  3. Appuyez sur « OK » dans la boîte de dialogue de confirmation

Remarque : Cette opération réapplique tous les plannings et paramètres de temps d'arrêt. Les paramètres de l'application sont conservés.

(iPhone/iPad) Peut-on empêcher la suppression de l'application Applimits ?

Vous pouvez utiliser la fonction Temps d'écran de l'iPhone/iPad pour empêcher la suppression d'Applimits :

  1. Ouvrez l'application « Réglages » sur votre iPhone
  2. Appuyez sur « Temps d'écran »
  3. Appuyez sur « Contenu et confidentialité »
  4. Appuyez sur « Achats iTunes et App Store »
  5. Réglez « Suppression d'apps » sur « Ne pas autoriser »

Recommandation : Définir un code de Temps d'écran offre une protection plus sûre. Faire gérer le code par un membre de la famille ou une personne de confiance est encore plus efficace.

Pour commencer

Qu'est-ce qu'Applimits ?

Applimits est une application pour gérer le temps d'utilisation de votre smartphone et vous aider à rester concentré. Vous pouvez définir des limites de temps pour des applications spécifiques, bloquer l'utilisation des applications pendant le coucher, et vous concentrer sur votre travail grâce à un minuteur Pomodoro.

Fonctionnalités principales :

  • Limite de temps d'utilisation : Définir le temps d'utilisation quotidien par jour de la semaine
  • Restreindre maintenant : Restreindre immédiatement les applications avec un minuteur
  • Restriction programmée : Définir des plages horaires où les applications ne peuvent pas être utilisées
  • Restriction stricte : Un mode puissant qui empêche la suppression immédiate des restrictions
Par où commencer pour la première utilisation ?

Étape 1 : Ouvrez l'application et suivez les instructions à l'écran pour autoriser les permissions Screen Time (Temps d'écran). Cette autorisation est obligatoire.

Étape 2 : Appuyez sur le bouton « + » de l'écran d'accueil pour créer un nouveau paramètre.

Étape 3 : Sélectionnez les applications ou catégories que vous souhaitez restreindre.

Étape 4 : Définissez les jours de la semaine et les limites de temps. Vous pouvez également utiliser des préréglages (jours ouvrables/week-end/tous les jours).

Étape 5 : Appuyez sur « Enregistrer » et c'est terminé.

Quelle est la différence entre Schedule et Downtime ?

Limite de temps d'utilisation : Définit un temps d'utilisation quotidien maximum pour des applications ou catégories spécifiques. Exemple : « Limiter les réseaux sociaux à 30 minutes par jour en semaine ».

Restriction programmée : Bloque complètement les applications sélectionnées pendant des plages horaires spécifiques. Exemple : « Bloquer les réseaux sociaux de 22h00 à 7h00 ». Idéal pour gérer l'heure du coucher.

Différences principales :

  • Schedule : Limite le temps d'utilisation total quotidien
  • Downtime : Blocage complet pendant des plages horaires spécifiques
Quelle est la différence entre la version gratuite et le plan Pro ?

Version gratuite :

  • Plannings : Jusqu'à 1
  • Restriction stricte : Non disponible

Plan Pro :

  • Plannings : Jusqu'à 6
  • Restriction stricte : Disponible

Essayez les fonctionnalités Pro avec un essai gratuit de 7 jours.

Limite de temps d'utilisation

Comment configurer les limites de temps d'utilisation ?

Vous pouvez définir le temps d'utilisation quotidien des applications.

Comment configurer :

  • Si plusieurs applications sont sélectionnées, la restriction s'applique lorsque le temps combiné dépasse le temps défini
  • Vous pouvez définir des limites de temps différentes pour chaque jour de la semaine
  • Utilisez les préréglages (jours ouvrables/week-end/tous les jours) pour une configuration groupée facile
  • En définissant 0 minute, les applications sont complètement bloquées ce jour-là

Exemple : Vous pouvez définir « 30 minutes lun-ven, 60 minutes sam-dim ».

Qu'est-ce que « Désactiver pour aujourd'hui » ?

La fonction « Désactiver pour aujourd'hui » vous permet de lever la restriction d'aujourd'hui et de la faire commencer à partir du lendemain.

Cas d'utilisation :

  • Vous avez besoin d'utiliser une application aujourd'hui pour des raisons particulières, mais vous souhaitez la restreindre à partir de demain
Que se passe-t-il lorsque la limite de temps est atteinte ?

Lorsque la limite de temps définie est atteinte, les applications ou catégories sélectionnées sont automatiquement bloquées.

Comportement pendant la restriction :

  • Lorsque vous essayez d'ouvrir une application restreinte, l'écran Temps d'écran de l'iPhone/iPad s'affiche
  • Si la « Restriction stricte » n'est pas activée, vous pouvez la lever temporairement avec « Désactiver pour aujourd'hui »
  • La limite de temps se réinitialise à 0h00 le lendemain et redevient active
(iPhone/iPad) Quelles applications sont concernées par la restriction par catégorie ?

Lorsque vous sélectionnez une catégorie, toutes les applications de cette catégorie sont automatiquement restreintes.

Exemples de catégories disponibles :

  • Réseaux sociaux (Twitter, Instagram, Facebook, etc.)
  • Divertissement (YouTube, Netflix, TikTok, etc.)
  • Jeux (toutes les applications de jeux)
  • Et bien d'autres catégories

Si vous installez une nouvelle application dans une catégorie, elle est automatiquement incluse dans la restriction.

Restriction programmée

Quand la restriction programmée est-elle la plus efficace ?

Cette fonction bloque complètement les applications pendant des plages horaires spécifiques. Elle est efficace dans les situations suivantes :

Heure du coucher : Bloquer les réseaux sociaux de 22h00 à 7h00 pour améliorer la qualité du sommeil

Temps d'étude : Bloquer les jeux et applications vidéo de 19h00 à 21h00

Prévention nocturne : Bloquer les applications de divertissement de 0h00 à 6h00 pour éviter de veiller tard

La restriction programmée peut-elle s'étendre au-delà de minuit ?

Oui, c'est possible. Vous pouvez configurer une restriction qui s'étend au-delà de minuit.

Exemple : En configurant « 22h00 – 7h00 », les applications seront bloquées de 22h00 jusqu'à 7h00 le lendemain matin.

Préréglages :

  • Heure du coucher (22h00 – 7h00)
  • Temps d'étude (19h00 – 21h00)
  • Nuit (0h00 – 6h00)

Vous pouvez également définir des paramètres différents pour chaque jour de la semaine.

Restreindre maintenant

Qu'est-ce que Restreindre maintenant ?

C'est une fonction qui vous aide à améliorer votre concentration en utilisant un minuteur et la technique Pomodoro.

Qu'est-ce que la technique Pomodoro : Une méthode de gestion du temps qui maintient une concentration élevée en alternant des sessions de travail concentré de 25 minutes avec des pauses de 5 minutes.

Comment l'utiliser :

  1. Sélectionnez les applications que vous souhaitez restreindre
  2. Définissez la durée du minuteur (15 minutes, 30 minutes, 1 heure, ou personnalisé)
  3. Activez le Pomodoro
  4. Définissez le temps de pause
Recevrai-je une notification à la fin du minuteur ?

Oui, vous recevrez une notification lorsque le minuteur se termine (si l'autorisation de notification est accordée).

Contenu de la notification :

  • Notification de fin du minuteur
  • Début du temps de pause défini

Remarque : L'autorisation de notification est facultative, mais sans elle, vous ne recevrez pas les notifications de fin de minuteur. Nous recommandons d'accorder l'autorisation de notification lors du premier lancement ou depuis l'écran des paramètres.

Restriction stricte

Qu'est-ce que la Restriction stricte ?

La Restriction stricte est une fonction de restriction puissante qui, une fois activée, ne peut pas être désactivée jusqu'à 0h00 le lendemain (plan Pro uniquement).

Caractéristiques :

  • Une fois activée, les paramètres ne peuvent pas être modifiés ou supprimés jusqu'à 0h00 le lendemain
  • Disponible pour les plannings, le minuteur de concentration et le temps d'arrêt
  • Renforce la maîtrise de soi et crée un environnement résistant aux tentations

Attention : Veuillez vérifier attentivement vos paramètres avant d'activer la Restriction stricte. Une fois activée, elle ne peut pas être annulée. (Si vous devez absolument la désactiver, veuillez supprimer l'application.)

La Restriction stricte ne peut-elle vraiment pas être désactivée ?

Oui, une fois la Restriction stricte activée, elle ne peut pas être désactivée jusqu'à 0h00 (minuit) le lendemain. C'est une conception intentionnelle.

Quand elle est levée :

  • Elle est automatiquement levée à 0h00 (minuit) le lendemain
  • Jusqu'à ce moment, elle ne peut pas être désactivée même en redémarrant l'appareil
  • Vous pouvez la désactiver en supprimant l'application, donc en cas d'urgence veuillez supprimer l'application
