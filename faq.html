ApLimitについての疑問を解決します

入門・基本

ApLimitとは何ですか？

ApLimitは、スマートフォンの使用時間を管理し、集中力を高めるためのiOSアプリです。特定のアプリやカテゴリに時間制限を設定したり、就寝時間中のアプリ利用をブロックしたり、ポモドーロタイマーで作業に集中することができます。

主な機能：

  • スケジュール管理：曜日や時間帯ごとに使用時間を制限
  • Downtime：就寝時間などにアプリをブロック
  • Focus Timer：ポモドーロテクニックで集中
  • Hard Limitation：制限を即座に解除できないモード
初めて使用する場合、何から始めればいいですか？

ステップ1： アプリを開き、画面の指示に従ってScreen Time（スクリーンタイム）の権限を許可します。

ステップ2： ホーム画面の「+」ボタンをタップして、新しいスケジュールを作成します。

ステップ3： 制限したいアプリまたはカテゴリを選択します。

ステップ4： 曜日と時間制限を設定します（プリセットも利用できます）。

ステップ5： 「保存」をタップして完了です。

ScheduleとDowntimeの違いは何ですか？

Schedule（スケジュール）： 特定のアプリやカテゴリに対して、曜日ごとに使用時間の上限を設定します。例えば「平日はSNSを1日30分まで」といった使い方ができます。

Downtime（ダウンタイム）： 特定の時間帯（例：22:00〜7:00）に、選択したアプリやカテゴリを完全にブロックします。就寝時間の管理に最適です。

無料版でできることは何ですか？

無料版では以下の機能が利用できます：

  • スケジュール作成：1つまで
  • アプリ選択：最大2つまで
  • Downtime設定：無制限
  • 基本的なスケジュール管理機能

より多くのスケジュールやアプリを管理したい場合は、Proプランへのアップグレードをご検討ください。

機能説明

Hard Limitation（厳格な制限）とは何ですか？

Hard Limitationは、一度有効にすると翌日まで無効化できない強力な制限機能です。自制心を強化したい場合に最適です。

特徴：

  • 有効にすると、その日の終わりまで設定を変更・削除できません
  • スケジュール、Focus Timer、Downtimeで利用可能
  • Proプラン限定の機能です

注意： Hard Limitationを有効にする前に、設定内容を十分に確認してください。一度有効にすると取り消せません。

Focus Timer（集中タイマー）の使い方は？

Focus Timerは、ポモドーロテクニックを使って集中力を高める機能です（Proプラン限定）。

使い方：

  1. Focus Timerタブを開く
  2. 制限したいアプリを選択
  3. タイマー時間を設定（15分、30分、1時間、またはカスタム）
  4. 必要に応じてHard Limitationを有効化
  5. 「開始」をタップ

タイマーが終了すると通知が届き、設定した休憩時間が始まります。

「今日を除外」とは何ですか？

「今日を除外」機能を使用すると、スケジュールの適用開始を翌日からに設定できます。

使用例：

  • 今日は特別な事情でアプリを使いたいが、明日からは制限したい
  • スケジュールを事前に準備しておきたい

スケジュールを再度有効にするには、設定画面の「スケジュールの再登録」を使用します。

アプリを個別ではなく、カテゴリで制限できますか？

はい、できます。アプリ選択画面では、個別のアプリだけでなく、カテゴリ全体を選択することも可能です。

利用可能なカテゴリ例：

  • ソーシャルネットワーキング
  • エンターテインメント
  • ゲーム
  • その他多数

カテゴリを選択すると、そのカテゴリに含まれるすべてのアプリが自動的に制限されます。

サブスクリプション・課金

Proプランには何が含まれますか？

Proプランでは以下の機能が利用できます：

  • スケジュール： 最大6つまで作成可能（無料版は1つ）
  • アプリ選択： 無制限（無料版は2つまで）
  • Focus Timer： ポモドーロタイマー機能
  • Hard Limitation： 厳格な制限モード

料金：

  • 月額プラン：¥480/月
  • 年間プラン：¥4,800/年（20%お得）

7日間の無料トライアルをお試しいただけます。

購入を復元するにはどうすればいいですか？

新しいデバイスに変更した場合や、アプリを再インストールした場合は、以下の手順で購入を復元できます：

  1. 設定画面を開く
  2. 「Proプランに登録」をタップ
  3. 「購入を復元」をタップ

購入時と同じApple IDでサインインしていることを確認してください。

サブスクリプションをキャンセルするには？

サブスクリプションはiPhoneの設定からキャンセルできます：

  1. iPhoneの「設定」アプリを開く
  2. 画面上部のあなたの名前をタップ
  3. 「サブスクリプション」をタップ
  4. 「ApLimit」を選択
  5. 「サブスクリプションをキャンセルする」をタップ

キャンセル後も、現在の請求期間が終了するまでProプランの機能は利用できます。

7日間の無料トライアル後はどうなりますか？

7日間の無料トライアル期間が終了すると、自動的に選択したプラン（月額または年間）での課金が開始されます。

トライアル期間中にキャンセルした場合は、課金されません。トライアル終了の24時間前までにキャンセルすれば、料金は発生しません。

トラブルシューティング

設定した制限が機能しません

制限が機能しない場合、以下を確認してください：

1. Screen Time権限の確認

  • iPhoneの「設定」→「スクリーンタイム」→「ApLimit」で権限が有効になっているか確認

2. スケジュールの設定を確認

  • 対象の曜日が有効になっているか
  • 時間制限が正しく設定されているか
  • 「今日を除外」が有効になっていないか

3. スケジュールの再登録を試す

  • 設定画面の「スケジュールの再登録」をタップ

それでも問題が解決しない場合は、お問い合わせフォームからご連絡ください。

スケジュールをリセットするにはどうすればいいですか？

スケジュールが正しく機能しない場合や、すべてをリセットしたい場合：

  1. 設定画面を開く
  2. 「スケジュールの再登録」をタップ
  3. 確認ダイアログで「OK」をタップ

注意： この操作により、すべてのスケジュールとDowntime設定が再適用されます。既存のスケジュール設定は保持されますが、iOS側の制限が再登録されます。

スケジュールを編集・削除できません

スケジュールを編集・削除できない主な理由：

Hard Limitationが有効になっている

  • Hard Limitationを有効にすると、その日の終わりまで変更できません
  • 翌日になれば編集・削除が可能になります

Focus Timerが実行中

  • タイマーが終了するまで待つ必要があります

これらは意図的な仕様で、自制心を強化するための機能です。

ApLimitアプリが削除されないようにできますか？

iOSのScreen Time機能を使用して、ApLimit自体の削除を防ぐことができます：

  1. iPhoneの「設定」アプリを開く
  2. 「スクリーンタイム」をタップ
  3. 「コンテンツとプライバシーの制限」をタップ
  4. 「iTunesおよびApp Storeでの購入」をタップ
  5. 「Appの削除」を「許可しない」に設定

スクリーンタイム・パスコードを設定すると、より確実に保護できます。

プライバシー・セキュリティ

ApLimitにはどんな権限が必要ですか？

ApLimitは以下の権限を必要とします：

Screen Time（スクリーンタイム）権限：必須

  • アプリの使用時間を制限するために必要です
  • ApLimitの全機能を使用するために必須の権限です

通知権限：オプション

  • Focus Timerの終了通知を受け取るために必要です
  • 許可しなくてもアプリは使用できますが、タイマー通知が届きません
ApLimitは私のデータをどのように扱いますか？

ApLimitはプライバシーを最優先に考えています：

  • すべてのデータはあなたのデバイスに保存されます
  • スケジュール設定やアプリ使用情報は外部に送信されません
  • サブスクリプション情報はAppleが管理します
  • 個人を特定できる情報は収集していません

詳細はプライバシーポリシーをご確認ください。
