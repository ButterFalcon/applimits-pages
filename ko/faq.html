문제 해결

설정한 제한이 제대로 작동하지 않습니다

⚠ iOS 26 사용자 안내

iOS 26에서 사용 시간 제한(앱 사용 시간 제한)이 정상적으로 작동하지 않는 버그가 보고되었습니다. 자세한 내용은 Apple Developer Forum을 참조하세요.

이 문제가 자주 발생하는 경우, 수정될 때까지 "예약 제한" 또는 "지금 바로 제한"의 사용을 고려해 주세요.

제한이 작동하지 않는 경우, 다음 사항을 확인하세요:

1. (iPhone/iPad 사용자) 기기 간 공유 비활성화

  • 스크린 타임을 다른 기기와 공유하는 경우, 동작이 불안정해질 수 있다고 Apple이 보고하고 있습니다
  • iPhone의 "설정" → "스크린 타임" → "기기 간 공유"를 비활성화하는 것을 권장합니다
  • Mac의 "설정" → "스크린 타임" → "기기 간 공유"를 비활성화하는 것을 권장합니다
  • iPhone 측 설정을 비활성화해도 Mac 측 설정이 활성화되어 있으면 "기기 간 공유"가 다시 활성화될 수 있으므로 모든 기기에서 비활성화하는 것을 권장합니다

2. (iPhone/iPad 사용자) Screen Time 권한 확인

  • iPhone의 "설정" → "스크린 타임" → "Applimits"에서 권한이 활성화되어 있는지 확인하세요
  • 권한이 비활성화된 경우, Applimits 앱에서 다시 허용해 주세요

3. (iPhone/iPad 사용자) MDM 사용자

  • Google Device Policy나 Intune 등의 기기 관리가 적용된 단말에서는 MDM 동기화 타이밍으로 인해 앱 내 사용 시간 설정에 문제가 발생할 수 있습니다
  • 문제가 발생한 경우, 설정 화면에서 "스케줄 재등록"을 실행해 주세요

4. 스케줄 재등록

  • 단말의 스크린 타임이나 기타 설정을 변경한 경우, 앱 설정이 해제될 수 있습니다.
  • 설정 화면의 "스케줄 재등록"을 탭하여 다시 설정해 주세요

5. 스케줄 설정 확인

  • 대상 요일이 활성화되어 있는지
  • 시간 제한이 0분 이외로 설정되어 있는지
  • "오늘만 비활성화"가 켜져 있지 않은지
OS 업데이트 후 제한이 작동하지 않습니다

OS 업데이트 후 제한이 작동하지 않는 경우, 다음을 시도해 주세요:

1. 기기 재시작

  • 기기를 완전히 재시작해 주세요

2. 스케줄 재등록

  • Applimits 설정 화면을 열기
  • "스케줄 재등록"을 탭
  • 모든 스케줄과 다운타임 설정이 다시 적용됩니다

3. Screen Time 권한 재확인

  • iPhone의 "설정" → "스크린 타임" → "Applimits"에서 권한을 확인하세요
"스케줄 재등록"이란 무엇인가요?

"스케줄 재등록"은 모든 스케줄과 다운타임 설정을 다시 적용하는 기능입니다.

사용 시기:

  • 제한이 제대로 작동하지 않는 경우
  • OS 업데이트 후

실행 방법:

  1. 설정 화면을 열기
  2. "스케줄 재등록"을 탭
  3. 확인 대화 상자에서 "확인"을 탭

참고: 이 작업으로 모든 스케줄과 다운타임 설정이 다시 적용됩니다. 앱 내 설정은 유지됩니다.

(iPhone/iPad) Applimits 앱이 삭제되지 않도록 할 수 있나요?

iPhone/iPad의 Screen Time 기능을 사용하여 Applimits 자체의 삭제를 방지할 수 있습니다:

  1. iPhone의 "설정" 앱을 열기
  2. "스크린 타임"을 탭
  3. "콘텐츠 및 개인 정보 보호 제한"을 탭
  4. "iTunes 및 App Store 구입"을 탭
  5. "앱 삭제"를 "허용 안 함"으로 설정

권장: 스크린 타임 암호를 설정하면 더 확실하게 보호할 수 있습니다. 가족이나 신뢰할 수 있는 사람에게 암호를 관리하게 하면 더욱 효과적입니다.

시작하기

Applimits란 무엇인가요?

Applimits는 스마트폰 사용 시간을 관리하고 집중력을 높이기 위한 앱입니다. 특정 앱에 시간 제한을 설정하거나, 취침 시간에 앱 사용을 차단하거나, 포모도로 타이머로 작업에 집중할 수 있습니다.

주요 기능:

  • 사용 시간 제한: 요일별로 하루 사용 시간 설정
  • 지금 바로 제한: 타이머 기능으로 즉시 앱 제한
  • 예약 제한: 앱을 사용할 수 없는 시간대 설정
  • 강력 제한: 제한을 즉시 해제할 수 없는 강력한 모드
처음 사용할 때 무엇부터 시작하면 되나요?

단계 1: 앱을 열고 화면의 안내에 따라 Screen Time(스크린 타임) 권한을 허용합니다. 이 권한은 필수입니다.

단계 2: 홈 화면의 "+" 버튼을 탭하여 새 설정을 만듭니다.

단계 3: 제한하고 싶은 앱 또는 카테고리를 선택합니다.

단계 4: 요일과 시간 제한을 설정합니다. 프리셋(평일/주말/매일)도 이용할 수 있습니다.

단계 5: "저장"을 탭하면 완료됩니다.

스케줄과 다운타임의 차이는 무엇인가요?

사용 시간 제한: 특정 앱이나 카테고리에 대해 하루 사용 시간의 상한을 설정합니다. 예: "평일에는 SNS를 하루 30분까지".

예약 제한: 특정 시간대에 선택한 앱을 완전히 차단합니다. 예: "22:00~다음 날 7:00에 SNS 차단". 취침 시간 관리에 최적입니다.

주요 차이점:

  • 스케줄: 하루 총 사용 시간을 제한
  • 다운타임: 특정 시간대에 완전 차단
무료 버전과 Pro 플랜의 차이는 무엇인가요?

무료 버전:

  • 스케줄: 1개까지
  • 강력 제한: 이용 불가

Pro 플랜:

  • 스케줄: 최대 6개까지
  • 강력 제한: 이용 가능

7일간의 무료 체험으로 Pro 기능을 사용해 보실 수 있습니다.

사용 시간 제한

사용 시간 제한은 어떻게 설정하나요?

하루 앱 사용 시간을 설정할 수 있습니다.

설정 방법:

  • 여러 앱을 선택한 경우 합산 사용 시간이 설정한 시간을 초과하면 제한이 적용됩니다
  • 요일별로 다른 시간 제한을 설정할 수 있습니다
  • 프리셋(평일/주말/매일)을 사용하면 일괄 설정이 간편합니다
  • 0분으로 설정하면 해당 요일에 완전히 차단됩니다

예: "월~금은 30분, 토·일은 60분"과 같은 설정이 가능합니다.

"오늘만 비활성화"란 무엇인가요?

"오늘만 비활성화" 기능을 사용하면 오늘의 제한을 해제하고 적용 시작을 다음 날부터로 설정할 수 있습니다.

사용 예:

  • 오늘은 특별한 사정으로 앱을 사용하고 싶지만, 내일부터는 제한하고 싶은 경우
시간 제한에 도달하면 어떻게 되나요?

설정한 시간 제한에 도달하면 선택한 앱이나 카테고리가 자동으로 차단됩니다.

제한 중 동작:

  • 제한된 앱을 열려고 하면 iPhone/iPad의 Screen Time 화면이 표시됩니다
  • "강력 제한"을 활성화하지 않은 경우, "오늘만 비활성화"로 일시적으로 해제할 수 있습니다
  • 다음 날 0:00에 시간 제한이 초기화되어 다시 활성화됩니다
(iPhone/iPad) 카테고리로 제한하면 어떤 앱이 대상이 되나요?

카테고리를 선택하면 해당 카테고리에 포함된 모든 앱이 자동으로 제한됩니다.

이용 가능한 카테고리 예:

  • 소셜 네트워킹 (Twitter, Instagram, Facebook 등)
  • 엔터테인먼트 (YouTube, Netflix, TikTok 등)
  • 게임 (모든 게임 앱)
  • 기타 다수의 카테고리

카테고리에 새 앱을 설치한 경우에도 자동으로 제한 대상에 포함됩니다.

예약 제한

예약 제한은 언제 사용하면 효과적인가요?

특정 시간대에 앱을 완전히 차단하는 기능입니다. 다음과 같은 상황에서 효과적입니다:

취침 시간: 22:00~다음 날 7:00에 SNS를 차단하여 수면의 질 향상

공부 시간: 19:00~21:00에 게임이나 동영상 앱을 차단

심야 대책: 0:00~6:00에 오락 앱을 차단하여 밤샘 방지

예약 제한은 날짜를 넘겨서 설정할 수 있나요?

네, 가능합니다. 날짜를 넘기는 설정이 가능합니다.

예: "22:00~다음 날 7:00"으로 설정하면 밤 10시부터 다음 날 아침 7시까지 앱이 차단됩니다.

프리셋:

  • 취침 시간 (22:00~다음 날 7:00)
  • 공부 시간 (19:00~21:00)
  • 심야 (0:00~6:00)

요일별로 다른 설정을 할 수도 있습니다.

지금 바로 집중

지금 바로 집중이란 무엇인가요?

타이머 기능이나 포모도로 테크닉을 사용하여 집중력을 높이는 기능입니다.

포모도로 테크닉이란: 25분의 집중 작업과 5분의 휴식을 반복하여 높은 집중력을 유지하는 시간 관리 기법입니다.

사용 방법:

  1. 제한하고 싶은 앱을 선택
  2. 타이머 시간을 설정 (15분, 30분, 1시간, 또는 사용자 지정)
  3. 포모도로를 활성화
  4. 휴식 시간을 설정
타이머 종료 시 알림을 받을 수 있나요?

네, 타이머가 종료되면 알림이 전송됩니다 (알림 권한을 허용한 경우).

알림 내용:

  • 타이머 종료 알림
  • 설정한 휴식 시간의 시작

참고: 알림 권한은 선택 사항이지만, 허용하지 않으면 타이머 종료 알림을 받을 수 없습니다. 최초 실행 시 또는 설정 화면에서 알림 권한을 허용하는 것을 권장합니다.

강력 제한

강력 제한이란 무엇인가요?

강력 제한은 한 번 활성화하면 다음 날 0:00까지 비활성화할 수 없는 강력한 제한 기능입니다 (Pro 플랜 전용).

특징:

  • 활성화하면 다음 날 0:00까지 설정을 변경하거나 삭제할 수 없습니다
  • 스케줄, 집중 타이머, 다운타임에서 이용 가능
  • 자제력을 강화하고 유혹에 지지 않는 환경을 만듭니다

주의: 강력 제한을 활성화하기 전에 설정 내용을 충분히 확인하세요. 한 번 활성화하면 해제할 수 없습니다. (부득이하게 해제가 필요한 경우 앱을 삭제해 주세요.)

강력 제한은 정말로 해제할 수 없나요?

네, 강력 제한을 활성화하면 다음 날 0:00(자정)까지 해제할 수 없습니다. 이것은 의도적인 설계입니다.

해제되는 시점:

  • 다음 날 0:00(자정)에 자동으로 해제됩니다
  • 그때까지는 기기를 재시작해도 해제할 수 없습니다
  • 앱을 삭제하면 해제할 수 있으므로 긴급 시에는 앱을 삭제해 주세요
