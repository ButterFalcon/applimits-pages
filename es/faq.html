Solución de problemas

Las restricciones configuradas no funcionan correctamente

⚠ Para usuarios de iOS 26

Se ha reportado un error en iOS 26 donde los límites de tiempo de uso (límites de app de Tiempo de pantalla) no funcionan correctamente. Para más detalles, consulte el Apple Developer Forum.

Si este problema ocurre con frecuencia, considere usar "Restricción programada" o "Restringir ahora" hasta que se corrija el error.

Si las restricciones no funcionan, verifique lo siguiente:

1. (Usuarios de iPhone/iPad) Desactivar Compartir entre dispositivos

  • Apple ha informado que compartir Tiempo de pantalla entre dispositivos puede causar inestabilidad
  • Recomendamos desactivar "Compartir entre dispositivos" en Ajustes de iPhone → Tiempo de pantalla
  • Recomendamos desactivar "Compartir entre dispositivos" en Ajustes de Mac → Tiempo de pantalla
  • Aunque desactive esta configuración en iPhone, si permanece activada en Mac, "Compartir entre dispositivos" puede reactivarse. Recomendamos desactivarlo en todos los dispositivos

2. (Usuarios de iPhone/iPad) Verificar permisos de Screen Time

  • Vaya a Ajustes de iPhone → Tiempo de pantalla → Applimits y verifique que los permisos estén activados
  • Si los permisos están desactivados, otórguelos nuevamente desde la aplicación Applimits

3. (Usuarios de iPhone/iPad) Usuarios de MDM

  • En dispositivos gestionados por MDM como Google Device Policy o Intune, la sincronización de MDM puede causar problemas con la configuración de tiempo de uso en la aplicación
  • Si ocurren problemas, ejecute "Volver a registrar horarios" desde la pantalla de configuración

4. Volver a registrar horarios

  • Si cambia la configuración de Tiempo de pantalla u otros ajustes del dispositivo, la configuración de la aplicación puede eliminarse.
  • Toque "Volver a registrar horarios" en la pantalla de configuración para reaplicar sus ajustes

5. Verificar la configuración del horario

  • ¿Están activados los días de la semana seleccionados?
  • ¿El límite de tiempo está configurado en algo diferente a 0 minutos?
  • ¿Está activada la opción "Desactivar solo por hoy"?
Las restricciones dejaron de funcionar después de una actualización del OS

Si las restricciones dejan de funcionar después de una actualización del OS, intente lo siguiente:

1. Reiniciar el dispositivo

  • Reinicie completamente su dispositivo

2. Volver a registrar horarios

  • Abra la pantalla de configuración de Applimits
  • Toque "Volver a registrar horarios"
  • Todos los horarios y configuraciones de tiempo de inactividad se reaplicarán

3. Verificar nuevamente los permisos de Screen Time

  • Vaya a Ajustes de iPhone → Tiempo de pantalla → Applimits y verifique los permisos
¿Qué es "Volver a registrar horarios"?

"Volver a registrar horarios" es una función que reaplica todos los horarios y configuraciones de tiempo de inactividad.

Cuándo usarlo:

  • Cuando las restricciones no funcionan correctamente
  • Después de una actualización del OS

Cómo ejecutarlo:

  1. Abra la pantalla de configuración
  2. Toque "Volver a registrar horarios"
  3. Toque "OK" en el cuadro de diálogo de confirmación

Nota: Esta operación reaplica todos los horarios y configuraciones de tiempo de inactividad. La configuración de la aplicación se conserva.

(iPhone/iPad) ¿Se puede evitar que la aplicación Applimits sea eliminada?

Puede usar la función de Tiempo de pantalla de iPhone/iPad para evitar que Applimits sea eliminada:

  1. Abra la aplicación "Ajustes" en su iPhone
  2. Toque "Tiempo de pantalla"
  3. Toque "Restricciones de contenido y privacidad"
  4. Toque "Compras en iTunes y App Store"
  5. Configure "Eliminar apps" en "No permitir"

Recomendación: Configurar un código de Tiempo de pantalla proporciona una protección más segura. Que un familiar o persona de confianza gestione el código es aún más efectivo.

Primeros pasos

¿Qué es Applimits?

Applimits es una aplicación para gestionar el tiempo de uso de su smartphone y ayudarle a mantenerse concentrado. Puede establecer límites de tiempo para aplicaciones específicas, bloquear el uso de aplicaciones durante la hora de dormir y concentrarse en su trabajo con un temporizador Pomodoro.

Funciones principales:

  • Límite de tiempo de uso: Establecer el tiempo de uso diario por día de la semana
  • Restringir ahora: Restringir aplicaciones inmediatamente con un temporizador
  • Restricción programada: Establecer franjas horarias en las que no se pueden usar las aplicaciones
  • Restricción estricta: Un modo potente que impide la eliminación inmediata de las restricciones
¿Cómo empiezo a usar la aplicación?

Paso 1: Abra la aplicación y siga las instrucciones en pantalla para otorgar los permisos de Screen Time (Tiempo de pantalla). Este permiso es obligatorio.

Paso 2: Toque el botón "+" en la pantalla principal para crear una nueva configuración.

Paso 3: Seleccione las aplicaciones o categorías que desea restringir.

Paso 4: Configure los días de la semana y los límites de tiempo. También puede usar preajustes (días laborables/fines de semana/todos los días).

Paso 5: Toque "Guardar" y listo.

¿Cuál es la diferencia entre Schedule y Downtime?

Límite de tiempo de uso: Establece un tiempo máximo de uso diario para aplicaciones o categorías específicas. Ejemplo: "Limitar las redes sociales a 30 minutos por día en días laborables".

Restricción programada: Bloquea completamente las aplicaciones seleccionadas durante franjas horarias específicas. Ejemplo: "Bloquear las redes sociales de 22:00 a 7:00". Ideal para gestionar la hora de dormir.

Diferencias principales:

  • Schedule: Limita el tiempo total de uso diario
  • Downtime: Bloqueo completo durante franjas horarias específicas
¿Cuál es la diferencia entre la versión gratuita y el plan Pro?

Versión gratuita:

  • Horarios: Hasta 1
  • Restricción estricta: No disponible

Plan Pro:

  • Horarios: Hasta 6
  • Restricción estricta: Disponible

Pruebe las funciones Pro con una prueba gratuita de 7 días.

Límite de tiempo de uso

¿Cómo configuro los límites de tiempo de uso?

Puede establecer el tiempo de uso diario de las aplicaciones.

Cómo configurar:

  • Si se seleccionan varias aplicaciones, la restricción se aplica cuando el tiempo combinado supera el tiempo establecido
  • Puede establecer diferentes límites de tiempo para cada día de la semana
  • Use preajustes (días laborables/fines de semana/todos los días) para una configuración masiva fácil
  • Configurar 0 minutos bloquea completamente las aplicaciones ese día

Ejemplo: Puede configurar "30 minutos de lunes a viernes, 60 minutos sábado y domingo".

¿Qué es "Desactivar solo por hoy"?

La función "Desactivar solo por hoy" le permite eliminar la restricción de hoy y hacer que comience a aplicarse desde mañana.

Caso de uso:

  • Necesita usar una aplicación hoy por razones especiales pero quiere restringirla desde mañana
¿Qué sucede cuando alcanzo el límite de tiempo?

Cuando alcanza el límite de tiempo establecido, las aplicaciones o categorías seleccionadas se bloquean automáticamente.

Comportamiento durante la restricción:

  • Al intentar abrir una aplicación restringida, se muestra la pantalla de Tiempo de pantalla del iPhone/iPad
  • Si la "Restricción estricta" no está activada, puede eliminarla temporalmente con "Desactivar solo por hoy"
  • El límite de tiempo se restablece a las 0:00 del día siguiente y vuelve a estar activo
(iPhone/iPad) ¿Qué aplicaciones se ven afectadas al restringir por categoría?

Al seleccionar una categoría, todas las aplicaciones de esa categoría se restringen automáticamente.

Ejemplos de categorías disponibles:

  • Redes sociales (Twitter, Instagram, Facebook, etc.)
  • Entretenimiento (YouTube, Netflix, TikTok, etc.)
  • Juegos (todas las aplicaciones de juegos)
  • Y muchas más categorías

Si instala una nueva aplicación en una categoría, se incluye automáticamente en la restricción.

Restricción programada

¿Cuándo es más efectiva la restricción programada?

Esta función bloquea completamente las aplicaciones durante franjas horarias específicas. Es efectiva en situaciones como:

Hora de dormir: Bloquear redes sociales de 22:00 a 7:00 para mejorar la calidad del sueño

Hora de estudio: Bloquear juegos y aplicaciones de video de 19:00 a 21:00

Prevención nocturna: Bloquear aplicaciones de entretenimiento de 0:00 a 6:00 para evitar trasnochar

¿La restricción programada puede extenderse más allá de la medianoche?

Sí, es posible. Puede configurarla para que se extienda más allá de la medianoche.

Ejemplo: Configurando "22:00 – 7:00", las aplicaciones se bloquearán desde las 10 PM hasta las 7 AM del día siguiente.

Preajustes:

  • Hora de dormir (22:00 – 7:00)
  • Hora de estudio (19:00 – 21:00)
  • Noche (0:00 – 6:00)

También puede establecer configuraciones diferentes para cada día de la semana.

Restringir ahora

¿Qué es Restringir ahora?

Es una función que le ayuda a mejorar la concentración utilizando un temporizador y la técnica Pomodoro.

¿Qué es la técnica Pomodoro? Un método de gestión del tiempo que mantiene una alta concentración alternando sesiones de trabajo concentrado de 25 minutos con descansos de 5 minutos.

Cómo usarlo:

  1. Seleccione las aplicaciones que desea restringir
  2. Configure la duración del temporizador (15 minutos, 30 minutos, 1 hora o personalizado)
  3. Active el Pomodoro
  4. Configure el tiempo de descanso
¿Recibiré una notificación cuando termine el temporizador?

Sí, recibirá una notificación cuando el temporizador termine (si se ha otorgado el permiso de notificaciones).

Contenido de la notificación:

  • Notificación de finalización del temporizador
  • Inicio del tiempo de descanso configurado

Nota: El permiso de notificaciones es opcional, pero sin él no recibirá notificaciones de finalización del temporizador. Recomendamos otorgar el permiso de notificaciones en el primer inicio o desde la pantalla de configuración.

Restricción estricta

¿Qué es la Restricción estricta?

La Restricción estricta es una función de restricción potente que, una vez activada, no se puede desactivar hasta las 0:00 del día siguiente (solo plan Pro).

Características:

  • Una vez activada, la configuración no se puede cambiar ni eliminar hasta las 0:00 del día siguiente
  • Disponible para horarios, temporizador de concentración y tiempo de inactividad
  • Fortalece el autocontrol y crea un entorno resistente a las tentaciones

Atención: Revise cuidadosamente su configuración antes de activar la Restricción estricta. Una vez activada, no se puede deshacer. (Si necesita desactivarla urgentemente, elimine la aplicación.)

¿Realmente no se puede desactivar la Restricción estricta?

Sí, una vez activada la Restricción estricta, no se puede desactivar hasta las 0:00 (medianoche) del día siguiente. Este es un diseño intencional.

Cuándo se desbloquea:

  • Se desbloquea automáticamente a las 0:00 (medianoche) del día siguiente
  • Hasta entonces, no se puede desactivar ni reiniciando el dispositivo
  • Puede desactivarla eliminando la aplicación, así que en caso de emergencia elimine la aplicación
